JS Functions Problem Set

The following problems are evaluated in the console using JS functions:

  1. Write a function isEven() which takes a single numeric arguement and returns true if the number is even, ad false otherwise
  2. Write a function factorial() which takes a single numeric arguement and returns the factorial of that number
  3. Write a function kebabToSnake() which takes a single kebab-cased string argument and returns the snake_cased version
