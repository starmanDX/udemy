JS Array Problem Set

Open the console to perform the following functions on arrays:

  • printReverse() - Individually prints the array elements, in reverse, to the console. EX: [1,2,3,4] prints 4,3,2,1 individually
  • isUniform() - Returns true if all numbers in an array are equal, or otherwise returns false. EX: [1,1,1,1] returns true | [1,1,1,3] returns false
  • sumArray() - Adds up all numbers in the array and returns the sum. EX: [2,4,6,8] returns 20
  • max() - Returns the largest number in an array. EX: [24,2,333,222] returns 333
